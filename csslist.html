利用可能なCSS一覧

SubmarinやMisskeyで使用可能なCSSの一覧です。

使い方

設定のクライアント設定欄にある『全般』メニューを開き、一番下にあるカスタムCSSを開いて使いたいCSSを貼り付けるだけです。

CSS 一覧

リアクションを見やすくする

by taiy / Midra

/* ノートに付いてる絵文字をちょっとだけ見やすくするやつ (@Midra@submarin.online) /
/ 他インスタンスの絵文字もちょっとだけ見やすくするやつ (@taiy@submarin.online) /
/ カスタマイズできるようにしたやつ (@taiy@submarin.online) /
/ Safariでの表示を修正したやつ (@Midra@submarin.online) /
/ 1:1以外の絵文字が潰れるのを修正したやつ (@taiy@submarin.online) /
/ 設定 /
/

  • 変数名 => 説明 (初期値)
  • –reactbtn-img => 画像の高さ (32px)
  • –reactbtn-gap => ボタン同士の幅 (4px)
  • –reactbtn-count => カウンターのmargin-right/left [6px <= (8px) <= 16px]
    /
    :root {
    –reactbtn-img: 32px;
    –reactbtn-gap: 4px;
    }
    /     ボタン全体 /
    .article > .main > .footer > div > button {
    display: inline-flex !important;
    align-items: center;
    padding: 0 !important;
    overflow: hidden;
    height: var(–reactbtn-img, 32px) !important;
    margin: 2px !important;
    }
    .article > .main > .footer > div > button:not(:last-child) {
    margin-right: var(–reactbtn-gap, 4px) !important;
    }
    .article > .main > .footer > div > button + button {
    margin-left: 0px !important;
    }
    .article > .main > .footer > div > button.canToggle {
    box-shadow: 1px 1px 2px 0px rgb(0 0 0 / 25%);
    }
    /     絵文字 /
    .article > .main > .footer > div > button > img.mk-emoji {
    min-width: var(–reactbtn-img, 32px) !important;
    height: 100% !important;
    background-color: #fff;
    padding: 4px;
    border-radius: 4px;
    box-sizing: border-box;
    }
    .article > .main > .footer > div > button.canToggle > img.mk-emoji {
    box-shadow: 0 0 2px 0px rgb(0 0 0 / 25%);
    }
    /     カウンター */
    .article > .main > .footer > div > button > img.mk-emoji + span {
    margin: 0 clamp(6px, var(–reactbtn-count, calc(var(–reactbtn-img, 32px) / 4)), 16px) !important;
    }

iOS14-16でノート外にコンテンツがはみ出す不具合の修正

by taiy

/* ノート外にコンテンツがはみ出る不具合の修正(iOS14,15,16) /
/ 破壊的変更・アイコンの追従表示不可*/
@supports not (overflow: clip) {
.notes > div,
.renote > div {
overflow: hidden !important;
}
.article > .avatar {
position: relative !important;
top: 0px !important;
}
}

画像の固定解除 (iOS14用)

by taiy

/* iOS14用・画像の固定解除 */
.files .gird-container::before {
content: none !important;
}
.files .gird-container > div {
position: relative !important;
}

ノートコンテンツの重なり優先度を下げる

by taiy

/* ノートのコンテンツの重なりの優先度を下げる /
/ Renote対応修正版 /
/ Reply対応修正版 */
.avatar {
z-index: 1 !important;
}
.header,
.footer {
position: relative !important;
z-index: 1 !important;
}
